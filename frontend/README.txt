Cara pakai (frontend):
1) Copy semua file di folder ini ke folder frontend project kamu.
2) Pastikan backend FastAPI jalan di http://localhost:8000
3) Buka index.html, login, lalu coba deteksi.

Catatan:
- File ini memakai localStorage untuk token & data sementara.
- Jika nanti hosting, ganti API_BASE di sadar-api.js sesuai URL backend.
